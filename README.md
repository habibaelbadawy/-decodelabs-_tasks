# Data Analysis Report

## Overview

The dataset contains 1200 online retail transactions with information about products, customers, payments, and order status. The objective was to analyze patterns, trends, and distributions within the data.

## Descriptive Statistics

Basic statistics including mean, median, and count were calculated for the numerical variables:

* Quantity
* UnitPrice
* ItemsInCart
* TotalPrice

These statistics provided an overview of customer purchasing behavior and transaction values.

## Trends Identified

* Product sales were analyzed to identify the most frequently purchased products.
* Payment methods were examined to determine customer preferences.
* Referral sources were analyzed to understand how customers reached the platform.
* Monthly sales trends were explored using transaction dates.

## Outlier Detection

Outliers were identified in the TotalPrice column using the Interquartile Range (IQR) method. These transactions represent unusually large purchases compared to the majority of orders.

## Key Findings

* The dataset contains 1200 transactions.
* Product purchasing patterns vary significantly across products.
* Certain payment methods are more popular than others.
* Referral sources contribute differently to customer acquisition.
* Monthly sales show fluctuations over time.
* A small number of high-value transactions were detected as outliers.

## Conclusion

The analysis provides useful insights into customer purchasing behavior, sales performance, and transaction distributions. The dataset is suitable for further business intelligence and predictive analytics tasks.
